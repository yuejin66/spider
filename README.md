#### 前言
1. 爬虫挺好玩的，兴趣学习，也能学到一些 HTTP 和分布式相关知识。
2. 主要用 Python HTTP 库或分布式爬虫框架 scrapy/scrapy-redis 进行采集信息。
3. 实际爬虫用什么编程语言都行，无论是 Go-HTTP 或 JAVA-HttpClient，这里用 python-Http 主要是方便快检，无需做过多处理，如果是 SpringBoot 项目，
   可以用 RPC 来做数据交换。
